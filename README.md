# simple-m3u8-player
A simple m3u8 player with buffer control and dynamic streaming.

The purpose of creating this library is to learn MSE details and hls playback process.



# Build and view demo

`npm install -g gulp gulp-cli live-server`

`npm install`

`npm run build`

`live-server`
